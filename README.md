# Cleanup 管理平台前端原型

基于当前项目 PRD、后台 UI 规范和现有业务链路整理出的静态 HTML 原型，可直接用于客户评审与后台流程演示。

## 预览入口

- 导航首页：`index.html`
- GitHub Pages 启用后，访问仓库 Pages 根地址即可打开导航页

## 页面范围

- 运营概览
- 用户管理
- 经销商管理
- 权限管理
- 内容配置
- 订单管理
- 物流节点模板
- 安装订单
- 订单详情
- 安装订单详情
- 施工单管理
- 施工单详情
- 检查模板
- 拍摄模板
- 验收模板

## 原型约定

- 技术形态：纯静态 HTML + CSS + 少量原生 JS
- 风格方向：Ant Design Pro 风格 PC 后台
- 用途：客户评审、业务梳理、后续 React/Vue 后台工程实现参考
- 当前数据：模拟数据，不接真实接口

## 推荐展示方式

- 仓库代码查看：适合客户查看页面目录与页面拆分
- GitHub Pages：适合客户直接点击页面进行原型评审

## 当前建议演示链路

- `index.html`
- `订单管理 -> 订单详情 -> 安装订单详情 -> 施工单详情`
