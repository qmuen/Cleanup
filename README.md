# 可丽娜管理平台前端原型

基于当前项目 PRD、后台 UI 规范和现有业务链路整理出的静态 HTML 原型。

## 预览入口

- 导航首页：`index.html`
- 推荐本地预览地址：`http://127.0.0.1:4182/index.html`

## 页面范围

- 运营概览
- 用户管理
- 经销商管理
- 权限管理
- 产品管理
- 内容配置
- 订单管理
- 物流节点模板
- 安装订单
- 订单详情
- 安装订单详情
- 施工单管理
- 施工单详情
- 检查模板
- 拍摄模板
- 验收模板

## 原型约定

- 技术形态：纯静态 HTML + CSS + 少量原生 JS
- 风格方向：Ant Design Pro 风格 PC 后台
- 用途：客户评审、业务梳理、后续 React/Vue 后台工程实现参考
- 当前数据：模拟数据，不接真实接口

## 最新产品管理口径

- 同步订单不携带图片，后台通过“产品管理”维护订单传入类型、产品名与产品图片、面板花色图片的映射关系
- 订单管理和订单详情展示图片时，从产品管理的映射关系中补齐展示素材
